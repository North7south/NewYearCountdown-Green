# 🎉 New Year Countdown Timer

A beautiful Chrome extension that counts down to New Year 2027 with elegant design and festive animations.

![New Year Countdown Timer](screenshots/promo_1.jpg)

## ✨ Features

- **Real-Time Countdown** - Accurate countdown to January 1st, 2027 midnight
- **Beautiful Design** - Soothing green gradient background with glass-morphism effects
- **Festive Animation** - Delicate fireworks animation adds celebratory atmosphere
- **Clean Interface** - Large, easy-to-read numbers with clear labels
- **Lightweight** - Fast loading with minimal resource usage
- **Zero Permissions** - Complete privacy, no data collection
- **No Configuration** - Install and enjoy immediately

## 🚀 Installation

### From Chrome Web Store

1. Visit the [Chrome Web Store](https://chrome.google.com/webstore) (link coming soon)
2. Click "Add to Chrome"
3. Click the extension icon in your toolbar to see the countdown

### Manual Installation (Developer Mode)

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/new-year-countdown.git
   ```

2. Open Chrome and navigate to `chrome://extensions/`

3. Enable "Developer mode" (toggle in top right)

4. Click "Load unpacked"

5. Select the extension directory

6. The extension icon will appear in your toolbar
